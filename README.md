## Building Backend Systems

A practical curriculum for learning backend engineering at the level of systems, not just frameworks.

This repository collects the course PDFs and supporting materials for the curriculum. The learning path follows the same progression described in the introduction: start with the network, then HTTP, APIs, databases, indexing, concurrency, authentication, caching, deployment, and AI integration.

### Who This Is For

This curriculum is for people who already know a little Python and want to understand what is happening underneath frameworks like FastAPI or Django. It is especially useful if you have built a small API, written database queries, copied authentication code from a tutorial, or deployed something without fully understanding the moving parts.

### What To Expect

- 46 days of guided material, one concept at a time
- A primary stack of Python, FastAPI, PostgreSQL, Redis, and Docker
- A capstone project called KasiCredit, an AI-assisted lending and credit scoring API for the African informal economy
- Theory, practice, and reflection in every day’s lesson
